O Netuno é desenvolvido em Java, então a partir do Java podemos criar novas funcionalidades.

Por exemplo, podemos criar novos serviços nativos, ou mesmo novos recursos de programação low-code que ficam
disponíveis em todas as linguagens de programação poliglota, como JavaScript, Python, Ruby, Groovy e Kotlin.

Em Java podemos também criar serviços web multi-aplicacional que ficam disponíveis em todas as aplicações do Netuno.

Qualquer técnica ou biblioteca do ecossistema do Java pode ser utilizada para se criar qualquer tipo de funcionalidade.

> Aconselhamos que antes de iniciar este tutorial você tenha algumas noções básicas de _Java_.
