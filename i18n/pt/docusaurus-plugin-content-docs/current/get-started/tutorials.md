# Tutoriais

Explore os tutoriais da plataforma Netuno e aprenda, na prática, como configurar recursos, modelar dados, criar interfaces e desenvolver lógica de negócio no backend.

Esta página reúne os principais conteúdos para quem está começando e também para quem deseja aprofundar o uso da plataforma. 

## Antes de começar

Antes de iniciar qualquer tutorial, verifique se seu ambiente está pronto:

- **Netuno instalado** e em execução no ambiente local ou de desenvolvimento.
- **Acesso ao painel administrativo** com permissões adequadas.
- **Conhecimentos básicos recomendados** sobre modelagem de dados, HTTP e APIs REST para aproveitar melhor os conteúdos técnicos.

Consulte o guia de [Instalação](./installation) antes de seguir.

## Ordem recomendada

Se esta é a sua primeira experiência com o Netuno, siga esta sequência:

1. Conheça a interface gráfica.
2. Aprenda a criar formulários e campos.
3. Configure relações entre formulários.
4. Avance para permissões e grupos de usuários.
5. Explore consultas e recursos de backend conforme a necessidade do projeto.

## Backend

O Netuno permite combinar low-code com desenvolvimento backend em diferentes linguagens e cenários. Os tutoriais abaixo ajudam você a criar integrações, manipular dados e implementar regras de negócio.

### Pré-requisitos

Antes de avançar para esta seção, é recomendado ter familiaridade com:

- Arquitetura cliente-servidor e protocolo HTTP.
- Estruturas de dados em JSON.
- Lógica de programação em JavaScript, Python ou outra linguagem compatível com seu fluxo de desenvolvimento.

:::tip Como escolher um tutorial

Use os tutoriais de **interface gráfica** quando o objetivo for configurar a aplicação visualmente, estruturar dados e administrar recursos pela interface.  
Use os tutoriais de **backend** quando for necessário criar lógica de negócio, integrações, autenticação, automações ou acesso programático aos dados.
:::
